# OneVC
OneVC is as CLI virtual cluster manager for [OpenNebula](http://opennebula.org).

## Requirement
* OpenNebula 3.0 and up
* Ruby Gems
    * Sequel

## Installation
    $ make install
    
## Uninstallation
    $ make uninstall
    
## Usage
    $ onevc <command> <options>
    
See onevc help text for more information
