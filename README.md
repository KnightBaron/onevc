# OneVC
OneVC is as CLI virtual cluster manager for [OpenNebula](http://opennebula.org).

## Requirement
* OpenNebula 3.0 and up
* Ruby Gems
    * sequel
    * rgl
    * daemons

## Installation
    $ make install
    
## Uninstallation
    $ make uninstall
    
## Usage
    $ onevc <command> <options>

## Issues
* Support only SQLite and MySQL (Untested) for now.
